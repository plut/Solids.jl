Quick-start

Basic example

using Solids
import Solids: Square, Circle, mult_matrix, translate, scale, color

union(
  color("pink")*
  translate([3,0])*
  scale([2,1])*
  Circle(3),

  color("cyan")*
  translate([0,5])*
  Square([2,3])
)

Solids.include — Function
	Solids.include(file::AbstractString, f::Function)

Reads given file and returns the union of all top-level Solids objects (except the results of assignments) found in the file.
